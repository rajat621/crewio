1) PROJECT OVERVIEW
==================================================
CrewControl is a workforce management platform with:
- Backend API (Node.js + Express + MongoDB/Mongoose)
- Frontend SPA (React + Vite + MUI)
- AI extraction service (Python FastAPI + Camelot)

Main business domains:
- Authentication and account security (OTP + optional 2FA)
- Employees and assignments
- Attendance tracking
- Company profiles
- Tax invoices and PDF generation
- File upload + extraction pipeline for invoice data

3) HIGH-LEVEL DATA FLOW
==================================================
Authentication Flow:
1. Frontend auth pages call backend auth endpoints.
2. Backend issues JWT.
3. Frontend stores token in localStorage (crewcontrol_token) and user object (crewcontrol_user).
4. API client attaches Bearer token automatically.

Protected API Flow:
1. Frontend calls API wrappers in crewcontrol-fron/src/api/*.js
2. Requests hit backend /api/* routes.
3. verifyToken middleware protects business routes.
4. Controllers call models/services and return JSON.

Invoice + AI Extraction Flow:
1. User uploads PDF via /api/upload.
2. Backend stores upload metadata and path.
3. Backend invoice service calls AI service /extract with PDF path.
4. AI service extracts rows from PDF tables.
5. Backend computes totals/tax, persists invoice, generates invoice PDF.
